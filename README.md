# Проект: Скрипт для парсинга и сортировки логов

## Цель
Создание скрипта `parse-logs.sh`, который находит файлы с расширением `.log`, валидирует их содержимое по заданной структуре и выводит логи в порядке убывания или возрастания по идентификатору `id`.

## Описание
Скрипт:
- Находит все `.log` файлы в указанной директории.
- Проверяет правильность формата логов.
- Сортирует и выводит логи по `id`.

### Формат логов
[<descriptor>]dd:mm:yyyy<id>_<message>

- `_` — пробел  
- `__` — пробельный символ (`\n`, `\r` и т.д.)  
- `dd:mm:yyyy` — дата (день-месяц-год)  
- `<descriptor>` — уровень логирования (`INFO`, `DEBUG`, `WARNING`, `ERROR`)  
- `<id>` — неотрицательное целое число — идентификатор лога  
- `<message>` — произвольное сообщение (может быть пустым)  

### Параметры скрипта
```bash
parse-logs.sh [OPTION]... DIR
```

DIR — директория с логами

Опции:

- `-l` --lines NUMBER — количество логов (по умолчанию 10)
- `-o` --order up|down — порядок сортировки (up — первые N, down — последние N; по умолчанию up)
